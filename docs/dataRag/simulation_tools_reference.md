# Technical Referance: In-Sim AI Tools

This document outlines the usage parameters for the tools available to users within the Edtronaut AI Coworker simulation. These tools can be invoked by the user or suggested by the Director agent.

## 1. The KPI Calculator
The KPI calculator is a utility that allows users to project the impact of new HR initiatives.
- **Inputs Required**: Baseline Metric (e.g., current retention rate), Proposed Change (percentage increase/decrease), Timeframe (in months).
- **Outputs**: Projected metric value, estimated cost savings, and a generic ROI score (1-100).
- **Usage Context**: The Gucci CHRO often demands hard numbers. Use the KPI Calculator to substantiate proposals regarding Inter-Brand Mobility.

## 2. Simple A/B Simulator
This tool simulates the reception of two different communication strategies or HR policies across the simulated employee base.
- **Inputs Required**: Strategy A Text, Strategy B Text, Target Audience (e.g., All EMEA Employees).
- **Outputs**: Simulated open rates, simulated positive/negative sentiment scores, and a recommended winner.
- **Usage Context**: The Employer Branding Manager should use this tool to navigate resistance when rolling out the new Competency Framework.

## 3. Portfolio Pack Export (One-Click)
This tool compiles the user's simulation performance into an employer-readable format.
- **What it does**: It aggregates the user's strategic plan, drafted internal posts, and executive updates (conversations with the CEO/CHRO).
- **Why it matters**: It serves as the tangible outcome of the Edtronaut platform. Users can take this exported PDF and showcase their actual problem-solving capabilities to real-world employers on LinkedIn or during interviews.
- **Guardrails**: The tool automatically strips out any confidential brand secrets and formats the output into a generic "Enterprise HR Strategy" portfolio piece.
